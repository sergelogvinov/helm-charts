# pgbouncer

![Version: 0.3.1](https://img.shields.io/badge/Version-0.3.1-informational?style=flat-square) ![Type: application](https://img.shields.io/badge/Type-application-informational?style=flat-square) ![AppVersion: 16.4](https://img.shields.io/badge/AppVersion-16.4-informational?style=flat-square)

Postgres connection poller

PgBouncer is a lightweight connection pooler for PostgreSQL.
It is designed to manage and optimize database connections by limiting the number of active connections to a PostgreSQL database, which helps improve performance and resource usage, especially in high-traffic environments.

```yaml
# helm values

userlist:
  # "username" "password"
  # "md5" + "md5(password + username)" // echo -n '1234admin' | md5 -> md545f2603610af569b6155c45067268c6b
  pgbouncer: password
  betmaster: md5$HASH

databases:
  betmaster:
    host: pg-0.pg-headless
    port: 5432
    user: username
    dbname: database
    poolmode: session

users:
  username2:
    poolmode: transaction

pgHbaConfiguration: |-
  hostssl all         all                   0.0.0.0/0       md5

serverSslMode: require
clientSslMode: require

updateStrategy:
  type: OnDelete

metrics:
  enabled: true
```
